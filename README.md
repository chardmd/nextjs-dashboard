## Next.js Dashboard

Dashboard Application built using Next.js

Covers the following:

- App Routers
- Layouts
- Data Fetching
- Server & Client Side Components
- Streaming
- Loading and Error UI
- Next-Auth
- Postgres serverless Setup
- Server Actions
- Validation and Error Handling
- Static Pages & SEO
